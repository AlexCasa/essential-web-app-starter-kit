# essential-web-app-starter-kit

Essential starter kit for web application.

## Usage

- Install dependencies: `npm install` or `yarn`

- Run dev: `gulp watch`

- Run dist: `gulp build`

## Conventions

The project must have the following structure:
```
  app/
    scripts/
    styles/
    images/
    server/
  public/
      ...
```
